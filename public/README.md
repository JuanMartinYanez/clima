# Fase 3 proceso de selección

Aplicación desarrolloda por: Ing. Juan Martín Yánez

## Uso

Inicialmente la aplicación al ser cargada muestra por defecto el clima de la ciudad de Cuenca, tanto para el día actual como para todos los 5 días previos y 7 posteriores.

Una vez cargada la aplicación el usuario debe selecionar una ciudad, un número de días anteriores y un número de días posteriores. Cuando se hayan seleccionado estos valores el usuario debe hacer click en general para visualizar los resultados.

La aplicación esta desplegada en el siguiente enlace: https://clima-743f5.web.app

## Descripción

Para el desarrollo de la aplicación de manera inicial se trabajo en la generación de elementos de diseño que aporten al aspecto visual de la misma, la mayoría de elementos gráficos utilizados estan en formato svg con el objetivo de que estos se presenten en la mejor calidad posible al usuario y buscando optimizar el tiempo de carga de los recursos.

La aplicación desarrollada es del tipo responsive, se utilizo el framework Bootstrap v4.3.1 (https://getbootstrap.com/).

Para desplegar la aplicación se utilizo el servicio de Hosting de Firebase, plataforma soportada por Google.
